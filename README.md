# Система управління студентами (Django)
Це веб-додаток, розроблений на базі фреймворку Django,
призначений для зручного управління базою даних студентів,
їхньою успішністю та контактною інформацією. Проєкт створено в рамках виконання лабораторної роботи.

Основний функціонал

- Список студентів: Перегляд повної таблиці студентів із зазначенням ПІБ, групи та оцінок з трьох предметів (Математика, Програмування, Англійська мова).
- Профілі: Можливість додавати та переглядати контактну інформацію (телефон/email) та фотографії студентів.
- Імпорт бази (CSV): Швидке завантаження списку студентів із локального `.csv` файлу.
- Імпорт з API: Автоматичне отримання та парсинг тестових даних студентів із зовнішнього відкритого API (JSONPlaceholder).
- Звіти та аналітика: Генерація інтерактивних стовпчикових діаграм (за допомогою Chart.js) для візуалізації успішності студентів.
- Адмін-панель: Повноцінне управління записами через вбудовану панель адміністратора Django.

Технології

- Backend: Python 3, Django
- Frontend: HTML, вбудована мова шаблонів Django, Chart.js для графіків
- База даних: SQLite за замовчуванням
- Робота з медіа: Pillow для обробки зображень
